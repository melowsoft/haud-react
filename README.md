# React contact app

## Summary

This project is a sample contact app for collecting users data, storing
editing and deleting

## Features

* Home screen has a table with users list
* Add user screen for adding new users
* Edit user screen for editing an existing user


## Installation
To run the app locally

- clone the repository locally with: 

```bash
  git clone https://github.com/melowsoft/haud-react.git
```
- cd into project directory with: 

```bash
  cd haud-react
```

- Install all dependencies in project directory with:

```bash
  yarn install
```

- To run the app locally:

```bash
  npm start
```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Testing

- To run test:

```bash
  npm run test
```
